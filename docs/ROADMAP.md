# Roadmap

## Phase 1 – Core (MVP)
- ls, pwd, cat, date, dirname, basename, realpath
- JSON output mode
- Library-first architecture

## Phase 2 – Text & FS
- grep (string-based)
- sort, uniq
- hash (sha256)
- archive (zip/tar)

## Phase 3 – Advanced
- Internal pipeline engine
- Diff (text + json)
- Watcher (fsnotify)

## Phase 4 – Ecosystem
- Taskfile linter
- Config/env handling
- Documentation & examples
