# Roadmap

## Phase 1 – Core (MVP) [DONE]
- `ls`, `pwd`, `cat`, `date`, `dirname`, `basename`, `realpath`
- JSON output mode (Partial)
- Library-first architecture

## Phase 2 – File Operations
- `cp` / `copy`, `mv` / `move`
- `rm` / `remove`, `rmdir`, `mkdir`
- `ln`, `readlink`
- `stat`, `touch`
- `chmod`, `chown`

## Phase 3 – Text Processing
- `grep`, `egrep`, `fgrep`
- `head`, `tail`, `cat` (enhanced), `tac`
- `sort`, `uniq`, `wc`, `nl`
- `cut`, `paste`, `join`, `fold`, `colum`
- `tr`, `sed`, `awk`

## Phase 4 – System & Information
- `env`, `whoami`, `id`
- `uname`, `uptime`, `time`
- `df`, `du`, `free`
- `ps`, `gops`, `top`, `kill`

## Phase 5 – Advanced Utilities & Flow
- `xargs`, `yes`, `nohup`
- `watch`, `less`, `more`
- `pipeline` (Internal engine)

## Phase 6 – Ecosystem
- Taskfile linter
- Config/env handling
- Documentation & examples
